# Zero-Cost AI Receipt Processor

## The Problem

Processing receipts manually is slow and error-prone. Traditional OCR is rigid and expensive.

## The Solution

A mobile-first, serverless web application using Gemini 2.5 Flash Vision for dynamic transaction parsing, combined with Firebase for instant ledger allocation.

## Tech Stack

- **Next.js** (App Router)
- **Tailwind CSS**
- **Firebase** (Auth + Firestore)
- **Google AI Studio** (Gemini 2.5 Flash)
